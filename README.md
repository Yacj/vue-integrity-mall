# mall
因学习和公司需求而开发的一个基于 vue+vant+axios的综合性商城,用esay-mock模拟数据

## 已解决的问题
- vant底部组件选中状态不会消失
- 通过localStorage来判断用户登陆状态
- 用vuex来显示购物车数量状态

## 待解决的问题
- 后续添加商品详情，用户主页等页面
- 编写Util.js，封装一些基本业务 如 url截取 Cookie等
- 根据业务需求添加新的功能
